Ana.Lytics es un proyecto personal de finanzas orientado a ayudar a las personas a organizar, entender y mejorar la gestión de su dinero de forma clara y visual.

Actualmente está desarrollado con HTML, CSS y SASS, con foco en una estructura limpia, diseño escalable y estilos reutilizables. El proyecto evolucionará incorporando JavaScript para interactividad y un backend con MongoDB para persistencia de datos.

La plataforma contempla:
- Un dashboard financiero interactivo
- Gestión de metas económicas
- Resumen histórico de ingresos y gastos
- Un apartado educativo para aprender conceptos clave de finanzas personales
- Un simulador de presupuesto que recomiende buenas prácticas según el comportamiento financiero del usuario

Ana.Lytics está en constante desarrollo y representa un proyecto integral que combina diseño frontend, lógica de programación y arquitectura backend aplicada a un problema real.
